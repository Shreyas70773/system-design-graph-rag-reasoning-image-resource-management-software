MASTER PROJECT PROMPT

Brand-Aligned Content Generation Platform - Capstone Implementation

You are building a **college capstone project** that demonstrates a graph-augmented content generation system. The system scrapes company websites, extracts brand data, and uses AI to generate brand-consistent marketing content.

PROJECT OVERVIEW
=================
Create a web application where clients can:
1. Enter their website URL → System scrapes company info and logo
2. Review extracted data → If logo quality is poor, offer AI enhancement
3. Add products/services → Via text description or product page URL  
4. Generate marketing content → AI creates images and copy that match their brand

TECHNICAL REQUIREMENTS
======================
- Budget: < $15/month (use FREE tier services only)
- No Docker required (local development + cloud hosting)
- Timeline: 8 weeks, 11 implementation phases
- Scale: 5-10 users for academic demonstration

DATA FLOW IMPLEMENTATION
========================
1. Web Scraper: BeautifulSoup extracts company name, tagline, logo, colors
2. Quality Checker: Pillow checks logo resolution/blur → score 0.0-1.0
3. Enhancement Options: If score < 0.7 → offer AI generation, upload, or keep
4. Product Parser: Groq Llama 3 extracts entities from text OR scrape product pages
5. Graph Storage: Neo4j stores Brand→Logo→Colors→Products relationships  
6. Content Generation: Build prompts with brand context → Hugging Face + Groq APIs
7. Brand Validation: Extract colors from generated images, compare to brand palette

# 1. Blur Detection (Laplacian Variance)
def detect_blur(image):
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY) 
    variance = cv2.Laplacian(gray, cv2.CV_64F).var()
    return variance  # < 100 = blurry
