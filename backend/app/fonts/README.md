# Fonts Directory

This directory should contain TTF font files for text overlay.

## Recommended fonts to download (free from Google Fonts):
- Montserrat-Bold.ttf
- PlayfairDisplay-Bold.ttf
- Roboto-Bold.ttf
- Poppins-Bold.ttf
- Oswald-Bold.ttf
- Lora-Bold.ttf
- Raleway-Bold.ttf
- BebasNeue-Regular.ttf

Download from: https://fonts.google.com/

The system will fall back to system fonts (Arial, DejaVu Sans) if these aren't available.
